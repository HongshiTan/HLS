## Setup to compile Vitis HLS LLVM
- Clone the sources
- Install [CMake 3.4.3 or higher](https://cmake.org/download/)
- Install [ninja](https://ninja-build.org/) [optional for faster builds]

## Compile Vitis HLS LLVM
run `build.sh`
